About eBC eBooks Collection

ebc generates a directory of electronic documents based on simple filename conventions.

The listed documents are ordered in three levels into categories, subcategories and topics. ebc recognizes the levels according to the following rules:
Categories are folders named ec_*, e.g. ec_Foundations.
Subcategories are subfolders of categories named es_*, e.g. ec_Foundations/es_Mathematics.
Topics are subfolders of categories or subcategories named et_*, e.g. ec_Foundations/es_Mathematics/et_Algebra.

ebc recognizes files with names beginning with eb_ or ex_.
ebc expects filenames according to the pattern
eb_Name, First name+_Title.ext,
with name and frist name of the author, several authors are joined by +, more authors not mentioned in the filename are indicated by a + behind the name or names.
Instead of the prefix eb_ one can use ex_. Files with names beginning with ex_ are treated as particularly interesting.
